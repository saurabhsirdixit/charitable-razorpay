== Description ==
razorpay is a free Payment Gateway for 5,00,000+ Businesses in India. There is no setup or annual fee. Just pay a transaction fee of 1.75%  for the transactions. razorpay accepts Debit Cards, Credit Cards, Net Banking, UPI, Wallets, EMI, NEFT and IMPS

[Charitable](https://wordpress.org/plugins/charitable/) plugin of the Wordpress allows non-profits and developers to create powerful fundraising platforms on their own website.

This plugin will help you integrate razorpay Payment Gateway with the Chartable Plugin without doing any efforts.

== Installation ==

1. Install the plugin from the [Wordpress Plugin Directory](https://wordpress.org/plugins/integrate-charitable-razorpay/).
2. To use this plugin correctly, you need to be able to make network requests. Please make sure that you have the php-curl extension installed.

== Dependencies == 

1. Wordpress v3.9.2 and later
2. Charitable v1.6 and later
3. PHP v5.6.0 and later
4. php-curl

== Configuration ==

1. Visit the Charitable settings page, and click on the Checkout/Payment Gateways tab
2. Click on Razorpay to edit the settings. If you do not see Razorpay in the list at the top of the screen make sure you have activated the plugin in the WordPress Plugin Manager
3. Do sign up, process will hardly take 10-15 minutes
[Sign Up on razorpay](https://dashboard.razorpay.com/#/access/signup)
4. During signup, razorpay will ask your PAN and Bank account details, after filling these details, you will reach razorpay Dashboard
5. On the left-hand side menu, you will see the option "Setting" click on this button
6. Choose the API key and generate the API Key
8. Copy "Key ID" & "Key Secret" and paste it in the Charitable razorpay extension setting
9. Save the settings and its done

For more details about razorpay service and details about transactions you need to access razorpay dashboard.
[Razorpay](https://razorpay.com/payment-gateway/)

**Note:** razorpay supports all currency at present. So this module will work with others currency.

== Frequently Asked Questions ==
= Does razorpay support currency other than INR? =

Yes, razorpay supports all currency.

= Does razorpay support International Payments? =

Yes, on the case to case basis. By default, International payments are disabled on new accounts. If you want to unable Internation payments for your account, you must have an FCRA certification, drop a mail to support@razorpay.com

= Is GST certificate mandatory for razorpay? =

It is good if you are having a GST certificate, but it is not mandatory.

= I am getting an error, "Payment Gateway Authorization Failed or API invalid" on the payment page, What should I do? =

Key ID and Key Secret are not the same. Make sure you uncheck "Turn on Test Mode" in Charitable plugin Payment Gateway settings for live mode. If you want to use test mode, please select test mode in rozarpay dashboard and get Test Keys from there.
